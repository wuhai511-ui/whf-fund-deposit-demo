# 资金预存产品演示

资金预存产品的交互式演示原型，用于展示商户资金预存、记账簿（Ledger）管理和交易核销的完整流程。

## 功能特性

- **📊 概览仪表盘**：实时展示总商户数、活跃消费者数、总预存金额
- **➕ 创建预存**：模拟消费者向商户预存资金的操作
- **📒 Ledger 管理**：查看所有消费者记账簿的余额和状态
- **📝 交易记录**：追踪每笔交易的金额、时间和状态
- **🔧 异常模拟**：模拟交易成功或失败的场景

## 技术栈

- HTML5
- CSS3（原生样式，无框架依赖）
- Vanilla JavaScript（无外部库）
- 数据完全存储在浏览器内存中（刷新后重置）

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可运行
2. 无需安装任何依赖或配置服务器

## 部署到 GitHub Pages

本项目已配置为可部署到 GitHub Pages：

1. 将代码推送到 GitHub 仓库的 `main` 分支
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为源
4. 访问生成的 URL 即可查看演示
